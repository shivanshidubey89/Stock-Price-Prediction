# 📈 Stock Price Prediction System using Machine Learning

A Machine Learning web application built using **Python, Scikit-Learn, Pandas, Matplotlib, and Streamlit** that predicts the **Closing Stock Price** based on market parameters such as Open Price, High Price, Low Price, and Volume.

---

## 🚀 Features

- 📊 Display Stock Dataset
- 🤖 Multiple Linear Regression Model
- 📈 Predict Closing Stock Price
- 📉 Model Accuracy (R² Score)
- 📊 Open Price vs Close Price Visualization
- 🎨 Interactive Streamlit Web Interface

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Streamlit

---

## 📂 Dataset

The dataset contains the following columns:

| Column | Description |
|---------|-------------|
| Open | Opening Price |
| High | Highest Price |
| Low | Lowest Price |
| Volume | Trading Volume |
| Close | Closing Price |

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/shivanshidubey89/Stock-Price-Prediction.git
```

Move into the project folder:

```bash
cd Stock-Price-Prediction
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run Stock.py
```
